# Future Features & Extensions

This document tracks features, enhancements, and ideas that are out of scope for the initial 45-day build but are valuable for future phases.

## Potential Extensions

- [ ] **Webhook Triggers:** Allow pipelines to be triggered by external webhooks with payload parsing.
- [ ] **Cron Scheduler UI:** A visual interface for managing scheduled pipeline runs.
- [ ] **Agent Memory:** Long-term memory for AI agents across execution runs.
- [ ] **Prompt Versioning:** History and rollback for prompt templates.
- [ ] **Template Marketplace:** A public or private hub for sharing workflow templates.
- [ ] **Multi-tenant Organizations:** Support for teams and organization-level permissioning.
- [ ] **Advanced RAG:** Reranking, hybrid search, and multi-modal retrieval.
